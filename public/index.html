SignUpEmail

Sends an OTP to the provided email for signing up.

Endpoint

POST /v1/email/signup-email

Request Body

{
  "email": "user@example.com"
}

Response

{
  "message": "OTP Send Successfully.",
  "status": 200
}

VerifySignUpOTP

Verifies the OTP for signing up. Deletes the OTP after verification.

Endpoint

POST /v1/email/verify-signup-otp

Request Body

{
  "email": "user@example.com",
  "otp": "123456"
}

Response

{
  "message": "OTP Verified Successfully.",
  "status": 200
}

Login

Logs in a user with the provided email and password.

Endpoint

POST /v1/auth/login

Request Body

    {
      "email": "user@example.com",
      "password": "password"
    }

Response

    {
      "message": "Login successful.",
      "data": {
        "_id": "user_id",
        "email": "user@example.com",
        "access_token": "jwt_access_token",
        "active_session_refresh_token": "refresh_token"
      }
    }

SignUp

Signs up a new user with the provided email, password, name, and phone number.

Endpoint

POST /v1/auth/customer/sign-up

Request Body

    {
      "email": "newuser@example.com",
      "password": "password",
      "name": "New User",
      "phone": "1234567890"
    }

Response

    {
      "message": "Customer Created Successfully.",
      "data": {
        "_id": "new_user_id",
        "email": "newuser@example.com",
        "name": "New User",
        "phone": "1234567890",
        "role": "Customer"
      }
    }
